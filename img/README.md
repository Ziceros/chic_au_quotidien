# Dossier images

Organisation conseillee pour les visuels de ChicAuQuotidien.

```text
img/
  logo.jpeg
  bannieres/
    accueil/
    categories/
    promotions/
  logos/
  icones/
  backgrounds/
  lookbook/
  temp/
  ceintures/
    enfants/
    femmes/
    hommes/
  chaussures/
    enfants/
    femmes/
    hommes/
  montres/
  sacs/
  vetements/
```

## Regles de nommage

- Utiliser des noms simples, sans espaces : `chaussure-homme-noir-01.jpeg`.
- Preferer les minuscules et les tirets `-`.
- Garder `temp/` uniquement pour les images en attente de tri.
- Placer les photos produits dans leur categorie, et les visuels du site dans `bannieres/`, `logos/`, `icones/`, `backgrounds/` ou `lookbook/`.
